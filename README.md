# 🎙️ GetVoice - Android 录音机

一款简单易用的 Android 语音录制 App，支持自定义文件名。

## ✨ 功能特性

- 🎤 **一键录音**：点击开始/停止，操作简单
- 📝 **自定义文件名**：录音完成后可重命名
- 📂 **公共目录保存**：文件保存在 Music/GetVoice 文件夹，方便查找
- 🎵 **MP3 格式**：高质量 AAC 编码，兼容性好
- 📱 **权限管理**：仅在需要时请求录音权限

## 📦 安装

1. 下载 [最新 APK](https://github.com/feier3388992/getvoice/releases)
2. 在手机上安装 APK
3. 打开 App，授予录音权限
4. 点击"开始录音"

## 🛠️ 自行编译

1. 克隆项目
2. 用 Android Studio 打开项目
3. 等待 Gradle 同步完成
4. Build -> Build APK
